# E&S Cancellation Return Premium Calculator

This repository contains a generic calculator for estimating return premium
amounts on cancelled excess and surplus lines policies. It is intended as a
neutral starting point for cancellation workflows and can be adapted to match
approved rating, filing, and compliance requirements.

## Purpose

The calculator is designed to help users estimate cancellation outcomes using
standard policy inputs such as:

- original premium
- policy effective date
- cancellation date
- policy expiration date
- earned premium basis
- applicable minimum earned premium
- fees or taxes that should be handled separately from premium

## Scope

This project is intentionally generic. It does not include carrier-specific
branding, insured information, policy numbers, screenshots, proprietary rules,
or internal references. Any production use should be reviewed against the
applicable policy form, state requirements, filing rules, and business approval
processes.

## Typical Workflow

1. Enter the policy term dates and cancellation date.
2. Enter the original premium and any applicable minimum earned premium.
3. Select the cancellation method supported by the implementation.
4. Review earned premium, unearned premium, and estimated return premium.
5. Validate the result before issuing any premium adjustment.

## Disclaimer

This calculator is for estimation and workflow support only. Final return
premium amounts should be confirmed against approved business rules, regulatory
requirements, and authorized policy documentation.
